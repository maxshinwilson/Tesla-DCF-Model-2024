# Tesla Discounted Cash Flow Valuation (2024 Base Year)

This project includes a fully integrated discounted cash flow (DCF) valuation of Tesla, Inc., using the financials of 2024 as the base year. The model focuses on conservative forecasting, transparency, and separation of historical data, assumptions, and valuation.

The purpose of this project is not to calculate a point-estimate of a price target, but to demonstrate valuation methodology and financial modeling.

---

## Files included
- `model/tesla_dcf_2024.xlsx` — Fully linked Excel DCF model  
- `images/` — Screenshots highlighting key sections of the model  
- `notes/` — Supporting documentation
